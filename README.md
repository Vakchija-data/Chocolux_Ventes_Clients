# Chocolux_Ventes_Clients
En tant qu'un Data Analyst Junior chez Chocolux, entreprise française de chocolats personnalisables haut de gamme, La mission est d’analyser les ventes des 4 derniers mois pour comprendre le comportement des clients, identifier les produits performants ou problématiques, et proposer des améliorations concrètes pour l’offre et le service
