# Contexte Professionnel – Chocolux
Tu viens d’intégrer Chocolux en tant que Data Analyst Junior, une entreprise française spécialisée dans les chocolats personnalisables haut de gamme, vendus via e-commerce et boutiques partenaires. La marque mise sur un positionnement premium : qualité des ingrédients, éthique de production et expérience client. Les chocolats sont noir, lait ou blanc et entièrement personnalisables, très prisés pour cadeaux d’entreprise, événements familiaux et fêtes (Noël, Pâques, Saint-Valentin), attirant une clientèle fidèle mais exigeante.

## Mission : analyser les données de ventes des 4 derniers mois à partir d’un fichier CSV contenant :
- Infos clients : âge, genre, ville
- Produits achetés : nom, type, prix, quantité
- Dates de commande
- Commentaires clients : positifs ou négatifs

## Objectifs :
- Comprendre le comportement des clients

- Identifier les produits les plus performants

- Détecter les signaux d’alerte (produits ou villes)

- Proposer des axes d’amélioration pour l’offre et le service
